All notable changes to the App Store Connect MCP Plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [0.0.1-alpha.0] - 2025-01-10

Initial alpha release of App Store Connect MCP server.

### Added
- 25 MCP tools for App Store Connect integration
- Pure Swift 6.0 implementation with actor-based architecture
- JWT authentication with App Store Connect API
- Firebase Crashlytics integration
- Certificate and provisioning profile management
- Bundle ID registration and capability management
- Build upload and validation via iTMSTransporter
- dSYM download and Firebase upload workflows
